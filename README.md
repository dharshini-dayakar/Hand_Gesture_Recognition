# ✋ Hand Gesture Recognition

A real-time hand gesture recognition system built using Python, OpenCV, MediaPipe, and Support Vector Machine (SVM).

## 📌 Features

* 👍 Thumbs Up Recognition
* 👎 Thumbs Down Recognition
* ✊ Fist Recognition
* ✌ Peace Recognition
* ✋ Palm Recognition
* Real-time webcam detection
* Fast gesture prediction using SVM

## 🛠 Technologies Used

* Python
* OpenCV
* MediaPipe
* Scikit-learn
* NumPy
* Streamlit

## 📂 Project Structure

```
Hand_Gesture_Recognition/
│── dataset/
│── collect_data.py
│── train_model.py
│── app.py
│── model.pkl
│── requirements.txt
│── README.md
```

## 🚀 Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
